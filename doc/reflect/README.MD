# 메타 프로그래밍

* 메타프로그래밍은 코드가 자신의 구조를 분석하고 수정할 수 있게 하는 프로그래밍 기법
* JavaScript에서 메타프로그래밍은 주로 Proxy와 Reflect 객체를 사용하여 객체의 동작을 변경하거나 제어





## Reflect

### 리플렉션이란?
런타임 시간에 객체에 변수 및 메소드를 조작하는 기능을 말합니다.
